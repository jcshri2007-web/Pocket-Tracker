# Pocket-Tracker
This is a Java-based Expense Tracker application designed to help users manage their finances efficiently. Users can input income and expenses, store data, and track their remaining balance over time. The application also provides visual insights through charts using Chart.js.

Features:

* Add and store income and expense details
* Calculates remaining balance using previous records
* Data persistence for continuous tracking
* Graphical visualization using Chart.js
* Simple and user-friendly interface

Tech Used:

* Java
* File Handling (for data storage)
* Chart.js (for visualization)
